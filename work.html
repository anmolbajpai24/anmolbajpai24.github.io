Tag Game

A hypercasual game based on the playground game tag that I made during my internship at Caravel Tech. This game is made using Unity and C#. I worked as the solo programmer on this game and collaborated with the design team to make it live! The ownership of the game belongs to Caravel Tech.

Anvesion - The Survival Game

Anvesion is a personal project which is currently under development. It is a survival game made with Unreal Engine and Blueprints in which the protagonist goes into the woods to discover the reason for numerous mysterious disappearances of people who tried to visit there. Along the journey he has to face multiple challenges in order to complete the game. Follow the development journey with devlogs of the game here

Photogrammetry of a Copper Bottle

I have an interest in bringing real life objects to the virtual world so I tried doing Photogrammetry of a Copper Bottle using the RealityScan App. The scan came out a bit distorted so I fixed it using Blender. Here's a link to the model.

Heart Model Labelling

A basic prototype to display the parts of a heart made using Physics Raycast and UI in Unity. This type of learning can be used in the Ed tech space in which children can interact with a model and learn about its different parts.

Choose The Colour

A Simple pick and drop game where you can pick up a color and it will be filled inside the cube, made for the better understanding of structs and singletons
